# Product Roadmap

This project is moving toward a local-first developer RPG layered on top of real coding activity.

## Phase 1: Live room prototype

Status: in progress

Delivered or partially delivered:

- local Codex transcript bridge
- live agent scene
- bottom utility rail
- GitHub-linked player profile
- commit-based level progression
- scene themes and office upgrade model

## Phase 2: Real progression loop

Focus:

- persistent upgrade ownership
- stronger coin economy
- more visible hardware progression
- better theme switching and room differentiation
- unlock feedback inside the office, not only inside the profile

## Phase 3: Project milestones

Focus:

- rewards tied to successful tests
- rewards tied to pull requests and merges
- session streaks
- notable project events such as releases and major refactors

The goal is to turn real development milestones into visible room growth.

## Phase 4: Shareability

Focus:

- recordable room snapshots
- public player cards
- office state sharing
- session highlight reels

This is where the project starts becoming inherently social and potentially viral.

## Phase 5: Multi-agent and multi-IDE support

Focus:

- support beyond Codex Desktop
- richer IDE integrations
- more specialized helper agents
- optional team rooms

The long-term vision is not a skin over one tool.
It is a broader interface layer for visible, playful software work.
